## Instructions for BotFather Setup

1. Open @BotFather in Telegram
2. Send `/mybots`
3. Select your bot
4. Choose "Edit Bot"

### Set Description:
- Click "Edit Description"
- Paste the "Bot Description (Short)" text above

### Set About:
- Click "Edit About"
- Paste the "About Text (Detailed)" text above

### Set Commands:
- Click "Edit Commands"
- Paste the "Bot Commands" text above

### Optional Settings:
- **Bot Picture**: Upload a graduation cap or university-themed icon
- **Inline Mode**: Disabled (not needed)
- **Privacy Mode**: Disabled (bot needs to read all messages)
